# Style Guide Enforcement Skill

## Instructions
Evaluate a document against the shared style guide and flag violations.

### Checks
- **Voice**: Active voice preferred. No passive constructions where active is possible.
- **Tone**: Professional, helpful, never condescending. Second person ("you") preferred.
- **Formatting**: Headers use ATX style. Code elements in backticks. Lists use dashes.
- **Terminology**: Consistent term usage throughout. No synonyms for the same concept.
- **Structure**: Required sections present and in the correct order.

### Output
List each violation with the file location and the specific rule being broken.
