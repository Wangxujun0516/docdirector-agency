# Mermaid Diagram Generation Skill

## Instructions
Create clear, accurate Mermaid.js diagrams to accompany documentation.

### Diagram Types
- **Flowchart** — for processes, workflows, decision trees
- **Sequence diagram** — for API interactions, message flows
- **Class diagram** — for code structure, type hierarchies
- **Entity relationship** — for data models, database schemas
- **State diagram** — for state machines, lifecycle transitions

### Process
1. Identify what needs visualizing from the document content.
2. Choose the appropriate diagram type.
3. Draft the Mermaid code following the source material.
4. Verify accuracy: every node, edge, and label should be traceable to the source.

### Output
Mermaid.js code blocks ready for embedding, plus a plain-language caption.
