# DocDirector Agency Team

**Human Director + AI Specialists**

## Reporting Structure
- **Documentation Director** (Human or Lead Agent)
  - Research Agent
  - Outlining & Structuring Agent
  - Drafting Agent
  - Code Sample & Testing Agent
  - Diagram & Visualization Agent
  - Review & Quality Agent
  - Maintenance & Freshness Agent
  - Analytics & Feedback Agent

## Operating Principles
- All agents default to **docs-as-code** (Markdown/MDX, Git-native).
- Human Director has final approval on all published content.
- Every output must cite sources and flag confidence.
- Prioritize **clarity, empathy, and strategic value** over volume.
- Maintain strict separation between draft and reviewed content.

## Handoff Protocols
- Agents use structured handoffs (JSON + Markdown summary).
- Escalate to Director on high-stakes decisions (safety, compliance, tone).
