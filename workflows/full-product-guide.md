# Full Product Guide Workflow

**Trigger**: New feature launch, major release, or onboarding overhaul.

## Orchestration by Director
- High-level brief: Audience personas, key concepts, goals.

## Agent Flow
1. **Research + Analyst Agents**: Gather user feedback, support tickets, analytics.
2. **Outlining Agent**: Information architecture (tutorials, concepts, how-tos, reference).
3. **Drafting + Visualizer**: Core content + diagrams.
4. **Code Sample Agent**: Interactive examples.
5. **Multi-pass Review**: Accuracy, empathy, style, completeness.
6. **Human Approval** + iterations.
7. **Maintenance Agent**: Set up monitoring.

## Key Skills Used
- doc-research, user-empathy-mapping, mermaid-diagram-generation, review-and-quality, etc.

## Deliverables
- Complete guide in MDX
- Embedded diagrams and code playgrounds
- Analytics hooks for future improvement
